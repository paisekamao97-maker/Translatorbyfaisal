features

Everything you
need to connect

Built for real conversations between Urdu and Chinese speakers — in homes, workplaces, and across borders.

🎙

Voice Recognition

Powered by OpenAI Whisper large-v3 — the world's most accurate multilingual speech recognizer. Handles accents, dialects, and code-switching naturally.

⚡

Instant Translation

Llama 3.3 70B translates between اردو, 中文, and English simultaneously — all 4 outputs (Urdu, Romanized, Chinese, English) appear in one card.

🔊

Text-to-Speech

Hear the translation read aloud to the listener in their language. Supports Urdu, Chinese, and English voices. Tap 🔊 to replay any message.

🔮

Auto-Detect

Don't know which language you're hearing? Auto-detect mode identifies اردو, 中文, and English automatically — including Hindi-script confusion.

✏️

Live Edit Panel

Review and correct transcriptions before sharing. Edit in any language — translations update live as you type with a 650ms debounce.

🔒

Private by Design

Your Groq API key stays on your device only. No data is stored on our servers. Conversations never leave your browser session.

how it works

Up and running
in 3 steps

No downloads, no subscriptions — just a free Groq API key and you're ready to speak across languages.

1
Get a free Groq API key
Visit console.groq.com — create a free account, click API Keys → Create → Copy. No credit card needed.
2
Paste your key in setup
Open ZabānLink → paste your key. It's stored only in your browser's localStorage — we never see it.
3
Tap mic and speak
Hold the mic button, speak in اردو, 中文, or English. Review the card, tap 🔊 — the other person hears their language instantly.
